JavaScript

JavaScript adalah bahasa pemrograman yang sering digunakan untuk mengembangkan aplikasi web interaktif. JavaScript merupakan bahasa scripting yang berjalan di sisi klien (client-side), artinya kode JavaScript dieksekusi di peramban web (browser) pengguna. JavaScript memungkinkan pengembang untuk menambahkan fungsionalitas dinamis ke halaman web, mengubah konten secara dinamis, berinteraksi dengan pengguna, dan berkomunikasi dengan server untuk mengambil atau mengirim data.

Adapun JavaScript yang digunakan dalam website ini yaitu meliputi :
1. Night Mode
2. Scroll Active Link
3. Show Scroll Up
4. Show and Hidden Menu
5. Shadow Header
6. Swiper Slide
